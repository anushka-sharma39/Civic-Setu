#civic-setu

CivicSetu is a lightweight, multilingual civic intelligence web platform designed specifically for Indian government bridging the gap between goverment and citizens .
It bridges the three most critical gaps in citizen government interaction: welfare scheme discovery, ward-level transparency, and complaint redressal all in a single, convenience first interface that works in (rightnow) Hindi,Punjabi,English.{aiming to add al 22 official language}, also will text to speech feature.

Built using modern web technologies (React 18, Vite, Tailwind CSS v4), CivicSetu requires no app download, works on any smartphone browser, and can be deployed at zero cost. The platform right now includes 25+ real government welfare schemes with intelligent filtering by age, income, gender, caste, BPL status, and disability — a combination no existing government portal currently offers at the ward level.
