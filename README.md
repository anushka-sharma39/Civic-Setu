

CivicSetu is a lightweight, multilingual civic intelligence web platform designed specifically for Indian government bridging the gap between goverment and citizens .
It bridges the three most critical gaps in citizen government interaction: welfare scheme discovery, ward-level transparency, and complaint redressal all in a single, convenience first interface that works in (rightnow) Hindi,Punjabi,English.{aiming to add al 22 official language}, also will text to speech feature.

Built using modern web technologies (React 18, Vite, Tailwind CSS v4), CivicSetu requires no app download, works on any smartphone browser, and can be deployed at zero cost. 

The platform right now includes 25+ real government welfare schemes with intelligent filtering by age, income, gender, caste, BPL status, and disability  a combination no existing government portal currently offers at the ward level. A real-time ward-level civic intelligence panel showing public works progress (with budget spend tracking), a governance directory with callable contacts, and an area alert system for road blocks, water disruptions, and weather advisories,etc . A structured complaint filing system with automatic department routing, complaint status tracking by reference number, and WhatsApp share functionality. Eliminates the need to know which department handles which issue.A city/ward selector onboarding screen, fixed dark/light sidebar navigation, three-language toggle (EN/HI/PA), notification bell with unread badges, and fully responsive mobile-first layout. 
CivicSetu uses a Single Page Application (SPA) architecture. This means the entire app loads once in the browser, and switching between modules happens instantly without reloading the page ; making it feel as fast as a native app. All state (which page is active, which language is selected, which ward is chosen) is stored in React's memory using the useState hook.
